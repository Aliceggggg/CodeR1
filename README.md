# CodeR1

============= Задание №1 =================
Ввод данных должен быть из файла
Было: данные (ёмкость рюкзака, количество артефактов, веса, ценности) вводились с клавиатуры через std::cin.
Стало: данные считываются из файла artifacts.txt с помощью std::ifstream input("artifacts.txt");.

Требуется проверка успешного открытия файла
Было: файл не использовался, проверка отсутствовала.
Стало: добавлена проверка if (!input.is_open()) { std::cout << "Не удалось открыть файл artifacts.txt\n"; return 1; }.

Требуется проверка корректности входных данных
Было: значения n и capacity не проверялись.
Стало: добавлена проверка if (n <= 0 || capacity <= 0) { std::cout << "Некорректные данные в файле.\n"; return 1; }.

Веса и ценности должны считываться из файла в контейнеры
Было: веса и ценности вводились по одному с клавиатуры в цикле.
Стало: добавлены векторы std::vector<int> weights(n); std::vector<int> values(n); и чтение из файла в циклах.

Сортировка предметов недопустима (ломает индексы)
Было: вызывалась insert_sort(weights, values);, что меняло порядок предметов.
Стало: вызов сортировки закомментирован как нарушающий условие задачи.

Необходимо использовать новую версию функции рюкзака с восстановлением решения
Было: вызывалась старая версия backpack, возвращающая только максимальную ценность.
Стало: вызывается новая версия backpack(capacity, weights, values, chosen_indices, total_weight);, которая возвращает также индексы выбранных предметов и суммарный вес.

Требуется выводить не только максимальную ценность, но и состав решения
Было: выводилась только максимальная возможная стоимость.
Стало: добавлен вывод номеров выбранных артефактов, суммарного веса и суммарной ценности:
«Максимальная ценность», «Выбранные артефакты», «Суммарный вес», «Суммарная ценность».

Подключён корректный заголовок для локали
Было: setlocale использовался без явного подключения <clocale>.
Стало: добавлен #include <clocale> для корректной работы setlocale.

Код разделён на модули
Было: вся логика (DP, восстановление решения, main) находилась в одном файле.
Стало: функции вынесены в Head6.h и Module6.cpp, а в Laba1.1.cpp остался только main и работа с вводом/выводом.
